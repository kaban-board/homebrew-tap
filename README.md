# Homebrew Tap

Custom Homebrew formulas by [@beshkenadze](https://github.com/beshkenadze).

## Installation

```bash
brew tap beshkenadze/tap
```

## Formulas

### Kaban

Terminal Kanban for AI Code Agents.

```bash
brew install kaban
```

Or directly:

```bash
brew install beshkenadze/tap/kaban
```
